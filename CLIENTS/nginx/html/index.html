TP Gestion des Logs - Zentech

Introduction

Bienvenue dans le TP de gestion des logs pour Zentech. Dans ce TP, vous apprendrez à configurer et utiliser la pile ELK (Elastic, Logstash, Kibana) pour analyser les logs de votre site web.

Objectifs du TP

L'objectif de ce TP est de :

  • Comprendre les bases de la gestion des logs
  • Installer et configurer Elastic, Logstash, et Kibana
  • Analyser les logs générés par le site web de Zentech

Étapes

  1. Étape 1 : Installation de la pile ELK
  2. Étape 2 : Configuration de Filebeat pour la collecte des logs
  3. Étape 3 : Analyse des logs dans Kibana
